# Client Side

### Dependencies 
```
npm install 
```
For all client side dependencies

### To Run The Server
```
npm start
```

# Server Side

### Dependencies
```
npm install
```
For all server side dependencies. But you have to install request-promise seperately.

```
npm install --save request-promise
```

### To Run The Server
```
npm start
```
